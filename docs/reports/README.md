## Как использовать шаблоны

- В каждом отчете уже есть:
  - цель и границы темы;
  - блоки с примерами кода/файлов для включения;
  - места под рисунки;
  - заготовки диаграмм в формате `mxGraphModel` (для draw.io / diagrams.net).
- Для рисунков рекомендуется создать подпапку `docs/reports/images/` и складывать туда PNG/SVG.
- Для исходников диаграмм рекомендуется создать подпапку `docs/reports/diagrams/` и сохранять XML.
